(Operators) infix ⊉

multi sub infix:<⊉>($a,$b --> Bool:D)

Not a superset of nor equal to operator.

Returns True if $a is not a superset of $b. Equivalent to !(>=).

say (1,2,3) ⊉ (2,3,1); # OUTPUT: «False␤» 
say (1,2,3) ⊉ (2,3);   # OUTPUT: «False␤» 
say 4 !(>=) (1,2,3);   # OUTPUT: «True␤»
