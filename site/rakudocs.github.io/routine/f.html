class IO::Path

From IO::Path

(IO::Path) method f

Defined as:

method f(IO::Path:D: --> Bool:D)

Returns True if the invocant is a path that exists and is a file. The method will fail with X::IO::DoesNotExist if the path points to a non-existent filesystem entity.
